# MarkDown CheatSheet

# Heading

# H1
## H2
### H3

```markdown
# H1
## H2
### H3
```

# Bold
**Bold**
```markdown
**bold text**
```

# Italic
*Italic*
```markdown
*italicaized text*
```

# Orderted List
1. First item
2. Second item
3. Third item
```markdown
1. First item
2. Second item
3. Third item
```

# Unordered List
- First item
- Second item
- Third item
```markdown
- First item
- Second item
- Third item
```
