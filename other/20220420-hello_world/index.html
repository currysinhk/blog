Hello World Hugo

To Start a Hugo local server

hugo server -D

To Create new markdown

hugo new xxx/xxx.md

To Build site

hugo

To change theme

  • Go to Hugo theme
  • Download zip from Github
  • Rename Zip from origin-hugo-theme-master to origin-hugo-theme
  • Add theme name to /config.toml
theme = "origin-hugo-theme"
